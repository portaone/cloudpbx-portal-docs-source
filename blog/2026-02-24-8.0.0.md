Release 8.0.0 brings new functionality and bugfixes. 
Minimal PortaSwitch version is **MR125** because the portal requires new API to access voicemails and voicemail transcriptions. Please consider updating to **MR125**.

### Enhancements:
- ICAP-2744: Playback auto-attendant prompts
- ICAP-2766: Allow users to access faxes 
- ICAP-2817: Allow users to select different call activity metrics
- ICAP-2916: Unable to update the extension’s name and number under the customer of a branch office
- ICAP-2933: Show Queued callback with position preservation
<!--truncate-->

### Bugfixes:

- ICAP-2798: Improve the precision of call activity reports with data aggregated by week and day
- ICAP-2802: Misaligned columns on the Phone book tab of a device profile
- ICAP-2814: Format of dates differs for "From Date/Time" and "Date/Time" in the call statistics report
- ICAP-2837: More actions icon misaligns with the other two icons for the list of extensions
- ICAP-2839: Adjust the style of the Save button for the SIP info modal
- ICAP-2896: Users cannot play back auto-attendant prompts
- ICAP-2898: Fax details do not fit the fax history dialogue
- ICAP-2899: Increase the height of the fax preview window
- ICAP-2901: Missing details about a selected voice message
- ICAP-2902: Time is displayed in different time zones on different pages
- ICAP-2904: Make the 'Phone number' field mandatory when creating directory entries
- ICAP-2905: Change the header for the fax details
- ICAP-2908: Issues with Call-recordings & Transcriptions on the Extension Call history page
- ICAP-2909: The counter of new voice messages overflows the badge
